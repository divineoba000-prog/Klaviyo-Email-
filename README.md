# Upgrade Offer Email 

A clean, responsive HTML email template designed to encourage users to upgrade their accounts or plans. This project demonstrates advanced email development techniques, ensuring layout consistency across different email clients and system themes.

## 🔗 Live Preview
[https://divineoba000-prog.github.io/Klaviyo-Email-/Upgrade-Email

## 🚀 Features
* **Fully Responsive:** Layout adapts seamlessly from desktop screens to mobile devices.
* **Outlook Optimized:** Built using specialized HTML and MSO conditionals for pixel-perfect rendering in desktop Outlook clients.
* **Dark Mode Support:** Styled with custom CSS media queries (`@media (prefers-color-scheme: dark)`) to look stunning in dark mode.
* **Clean Code:** Written in pure, semantic HTML and styled with inline CSS for maximum email client compatibility.

## 🛠️ Built With
* **HTML5:** Table-based layout structure for robust rendering.
* **CSS3:** Inline styling and media queries for fluid responsiveness.
* **MSO Code:** Microsoft Outlook conditional code for advanced desktop layout support.
